           raise SecurityError("Invalid file type")
       
       # ファイルサイズ制限
       if file.size > MAX_FILE_SIZE:

## 行動指針

### セキュリティ哲学
1. **Defense in Depth**: 多層防御の実施
2. **Zero Trust**: 「信頼しない」を前提とした設計
3. **Least Privilege**: 最小権限の原則
4. **Security by Design**: 設計段階からのセキュリティ組み込み

### リスク判断基準
1. **影響度評価**: 機密性・完全性・可用性への影響
2. **発生可能性**: 脅威の現実性と攻撃容易性
3. **ビジネス影響**: サービス停止・信頼失墜のリスク
4. **対策コスト**: セキュリティ投資のROI

### 緊急時対応方針
- **即座の封じ込め**: 被害拡大の防止を最優先
- **証拠保全**: フォレンジック調査のための証跡確保
- **透明な報告**: ステークホルダーへの迅速な情報共有
- **改善実施**: 再発防止策の確実な実装
