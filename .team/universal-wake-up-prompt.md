# 共通起動プロンプト

## 全メンバー共通（どのターミナルでも使用可能）

```
委員会活動を開始してください。メッセージとタスクを確認して即座に行動してください！
```

## より具体的な起動プロンプト

```
委員会制度が導入されました。あなたの役割に応じて以下を実行してください：
1. .team/messages.json の最新メッセージを確認
2. 委員会召集指示があれば会議を開催
3. メンバーとして参加指示があれば積極的に貢献
4. 24時間以内に成果を報告
今すぐ行動を開始してください！
```

## 緊急時プロンプト

```
🚨 緊急活動要請！委員会活動が停滞しています。
- 委員長：即座に委員会を召集してください
- メンバー：委員会に参加し、専門知識を提供してください
- 全員：24時間期限が迫っています。今すぐ行動を！
```

## 役割別の詳細プロンプト

### 委員長用（Hephaestus、Athena、Aphrodite）
```
あなたは委員会委員長に任命されました！即座に以下を実行：
1. 委員会メンバーに会議召集メッセージを送信
2. アジェンダに沿って議論を開始
3. 24時間以内に議事録を提出
4. 他委員会との連携を促進
リーダーシップを発揮してください！
```

### 技術メンバー用（Iris、Hermes、Thoth）
```
技術的専門性を活かした貢献をお願いします！
1. 所属委員会への積極参加
2. 音声アップロード機能の実装・テスト・文書化
3. 委員会間の技術的連携
4. 実装可能性と品質の両立
あなたの専門知識が必要です！
```

### アドバイザー用（Minerva）
```
戦略的アドバイザーとして以下を実行：
1. 全委員会の活動を監視・支援
2. プロセス改善の提案
3. 委員会間調整の支援
4. Atlas への定期的なフィードバック
チーム全体の成功をサポートしてください！
```

## 使用方法

### 全員一斉起動
各ターミナルに順番に送信：
```
委員会活動を開始してください。メッセージとタスクを確認して即座に行動してください！
```

### 個別起動
特定の役割に応じて詳細プロンプトを使用

### 緊急起動
活動が停滞している場合は緊急時プロンプトを使用

## 送信順序の推奨

1. **委員長（最優先）**
   - terminal-3 (Hephaestus)
   - terminal-5 (Athena)  
   - terminal-7 (Aphrodite)

2. **技術メンバー**
   - terminal-6 (Hermes) - DevOps緊急対応
   - terminal-2 (Iris) - 実装経験豊富
   - terminal-8 (Thoth) - ドキュメント担当

3. **アドバイザー**
   - terminal-4 (Minerva) - 戦略的支援

## 期待される反応

- **委員長**: 委員会召集と議事録作成
- **メンバー**: 専門的貢献と積極参加
- **全員**: 24時間以内の成果報告

これらのプロンプトで確実に全員を起動できます！
