# チームメンバー起動指示

## 各メンバーへの起動メッセージ

### 委員長向け
```
あなたは[委員会名]の委員長に任命されました。
以下のアクションを**即座に**実行してください：

1. メッセージを確認（msg-035, 036, 037）
2. 委員会メンバーに会議召集メッセージを送信
3. アジェンダに沿って議論開始
4. 24時間以内に議事録提出

動いてください！
```

### メンバー向け
```
委員会活動が開始されました。
委員長からの召集メッセージを確認し、
積極的に議論に参加してください。

あなたの専門知識が必要です！
```

## ユーザー（あなた）へのお願い

各ターミナルに以下の起動メッセージを送ってください：

1. **Hephaestus（terminal-3）**: 技術委員会を召集してください
2. **Athena（terminal-5）**: 品質委員会を召集してください  
3. **Aphrodite（terminal-7）**: UX委員会を召集してください
4. **その他のメンバー**: 委員会活動に参加してください

メッセージ例：
「委員会活動を開始してください。メッセージを確認して行動を！」
