## 🎵 音声アップロード機能について

音声ファイル（MP3・WAV形式）をアップロードして、自動で文字起こしを行う機能です。会議録音や講演音声を簡単にテキスト化できます。

## 🚀 クイックスタート（3分で開始）

### ステップ1: 音声ファイルを準備
- **対応形式**: MP3、WAV
- **最大サイズ**: 100MB
- **推奨品質**: 44.1kHz、16bit以上

### ステップ2: アップロード画面を開く
1. システムにログイン
2. 新しいセッションを作成
3. 「音声アップロード」タブを選択

### ステップ3: ファイルをアップロード
1. ファイルをドラッグ&ドロップ、または「ファイル選択」ボタンをクリック
2. 音声ソースを選択（NOTTA または MANUS）
3. 「アップロード開始」ボタンをクリック

### ステップ4: 処理完了を待つ
- リアルタイムで進捗が表示されます
- 10分の音声で約90秒の処理時間
- 処理中はブラウザを閉じないでください

## ⚠️ エラーと対処法

### よくあるエラー

#### ファイル形式エラー
```
❌ 対応していないファイル形式です
   MP3またはWAVファイルをアップロードしてください

対処法:
・音声ファイルをMP3またはWAV形式に変換
・拡張子が正しいか確認
```

#### ファイルサイズエラー
```
❌ ファイルサイズが大きすぎます
   100MB以下のファイルをアップロードしてください

対処法:
・音声を分割して複数回アップロード
・音質を下げてファイルサイズを縮小
・不要な無音部分をカット
```

#### ネットワークエラー
```
❌ アップロードに失敗しました
   ネットワーク接続を確認してください

対処法:
・インターネット接続を確認
・[再試行] ボタンをクリック
・ブラウザを再読み込み
```

#### 処理タイムアウト
```
❌ 処理時間が超過しました
   音声ファイルを短く分割してください

対処法:
・10分以下の音声に分割
・音質を下げてファイルサイズを縮小
・時間をおいて再度お試しください
```

### 接続エラーの対処

#### WebSocket接続エラー
```
❌ リアルタイム更新に失敗しました

自動対処:
・3回まで自動再接続を試行
・ページを更新せずお待ちください

手動対処:
・[再接続] ボタンをクリック
・ブラウザを再読み込み
```

## 🎯 最適な使用方法

### 音声ファイルの準備

#### 推奨設定
- **形式**: WAV (高品質) または MP3 (サイズ重視)
- **サンプリングレート**: 44.1kHz以上
- **ビット深度**: 16bit以上
- **音声長**: 10分以下に分割推奨

#### 音質向上のコツ
1. **静かな環境での録音**: 背景ノイズを最小限に
2. **マイクとの距離**: 30cm程度を保つ
3. **一定の音量**: 急激な音量変化を避ける
4. **明瞭な発話**: ゆっくり、はっきりと話す

### 効率的な利用法

#### 長時間音声の処理
```
60分の会議録音の場合:

❌ 悪い例:
・60分を一度にアップロード
・処理時間: 約9分、エラーリスク高

✅ 良い例:  
・10分×6ファイルに分割
・処理時間: 各90秒、並行処理で効率的
```

#### 複数ファイルの処理
- **同時処理**: 最大3ファイルまで並行処理可能
- **待機中の活用**: 他のファイルを準備
- **結果確認**: 完了順に文字起こし結果を確認

## 📊 処理時間の目安

| 音声長 | ファイルサイズ | 処理時間 |
|--------|---------------|----------|
| 1分    | 1-2MB        | 約15秒   |
| 5分    | 5-10MB       | 約45秒   |
| 10分   | 10-20MB      | 約90秒   |
| 20分   | 20-40MB      | 約3分    |

※ネットワーク環境とサーバー負荷により変動します

## 🔒 セキュリティとプライバシー

### データ保護
- **転送時暗号化**: TLS 1.3による安全な通信
- **一時保存**: 処理完了後24時間で自動削除
- **アクセス制限**: セッション所有者のみアクセス可能

### プライバシー配慮
- **音声内容**: サーバーに永続保存されません
- **個人情報**: 文字起こし結果のみ保存
- **第三者アクセス**: 他のユーザーは閲覧不可

## 🛠️ トラブルシューティング

### システム要件
- **ブラウザ**: Chrome 90+、Firefox 88+、Safari 14+
- **JavaScript**: 必須（有効にしてください）
- **インターネット**: 安定した接続（1Mbps以上推奨）

### パフォーマンス最適化
1. **ブラウザキャッシュクリア**: 定期的に実行
2. **他のタブを閉じる**: メモリ使用量を削減
3. **VPN使用時**: 接続が不安定な場合は一時的に無効化

### サポート連絡先
問題が解決しない場合：
1. エラーメッセージをスクリーンショット
2. 使用ブラウザとバージョンを確認
3. 音声ファイルの詳細情報（形式、サイズ、長さ）
4. システム管理者にお問い合わせください
