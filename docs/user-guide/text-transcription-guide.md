# テキストベース議事録システム 利用ガイド

## 概要

このシステムは、事前に作成されたテキストファイルから議事録を作成・管理するためのツールです。NOTTAやManusなどで生成された文字起こしデータを効率的に編集・管理できます。

## 🚀 クイックスタート（5分で開始）

### 1. セッション作成
1. ホーム画面で「新しいセッション」をクリック
2. セッション名と開催日を入力
3. 「作成」ボタンをクリック

### 2. ファイルアップロード
1. 作成したセッションを開く
2. 「ファイルアップロード」タブを選択
3. NOTTAデータまたはManusデータをドラッグ&ドロップまたはクリックで選択
4. アップロード完了を待つ

### 3. データ編集
1. 「データ比較」タブに切り替え
2. 議事録の内容を編集
3. 不要なセクションを除外
4. 話者名やタイムスタンプを修正

### 4. エクスポート
1. 編集完了後、「選択したセクションをダウンロード」をクリック
2. テキスト形式またはWord形式を選択
3. ファイルをダウンロード

## 📁 対応ファイル形式

### アップロード可能な形式
- **テキストファイル（.txt）**: プレーンテキスト形式
- **Wordファイル（.docx）**: Microsoft Word形式

### ファイルサイズ制限
- **最大ファイルサイズ**: 10MB
- **推奨サイズ**: 1MB以下（処理速度向上のため）

### ファイル内容の推奨形式

#### NOTTAデータの例
```
0:00 田中: 皆さん、おはようございます。今日は重要な会議です。
0:15 佐藤: よろしくお願いします。
0:30 田中: まず、前回の議題から確認しましょう。
```

#### Manusデータの例
```
【1】開会挨拶 [田中] [9:00]
皆さん、おはようございます。今日は重要な会議です。

【2】前回議題の確認 [田中] [9:05]
まず、前回の議題から確認しましょう。
```

## 🔧 主な機能

### 1. ファイルアップロード
- **ドラッグ&ドロップ対応**: ファイルを直接ドロップしてアップロード
- **ファイル検証**: 形式とサイズを自動チェック
- **エラーハンドリング**: 分かりやすいエラーメッセージと解決方法を表示

### 2. セクション管理
- **自動セクション番号付与**: アップロード時に自動的にセクション番号を追加
- **セクション除外機能**: 不要なセクションを除外して出力
- **話者名編集**: 発言者名を修正・統一可能
- **タイムスタンプ編集**: 時刻情報の修正が可能

### 3. データ比較
- **サイドバイサイド表示**: NOTTAとManusデータを並べて比較
- **リアルタイム編集**: その場で内容を編集可能
- **変更の自動保存**: 編集内容は自動的に保存

### 4. エクスポート機能
- **テキスト形式**: プレーンテキストでの出力
- **Word形式**: Microsoft Word文書での出力
- **選択的出力**: 除外したセクションを除いて出力
- **時間計算**: 含まれるセクションの合計時間を計算

## 📝 使用方法詳細

### ファイルアップロード手順

1. **セッションページを開く**
   - ホーム画面からセッションを選択
   - 「ファイルアップロード」タブが選択されていることを確認

2. **NOTTAデータのアップロード**
   - 左側の「NOTTAデータ」セクションにファイルをドラッグ&ドロップ
   - または「クリックして選択」でファイルを選択
   - アップロード中は進捗が表示されます

3. **Manusデータのアップロード**
   - 右側の「Manusデータ」セクションに同様の手順でアップロード
   - 両方のファイルがアップロードされると自動的に比較画面に切り替わります

### データ編集手順

1. **データ比較画面に移動**
   - 「データ比較」タブをクリック
   - NOTTAとManusのデータが並んで表示されます

2. **内容の編集**
   - Manusデータの内容をクリックして編集
   - 発言者名、タイムスタンプ、内容を修正可能
   - 変更は自動的に保存されます

3. **セクションの除外**
   - 不要なセクションの「除外」チェックボックスをチェック
   - 除外されたセクションは出力時に含まれません

4. **時間計算**
   - 「合計時間を計算」ボタンをクリック
   - 除外されていないセクションの合計時間が表示されます

### エクスポート手順

1. **出力形式の選択**
   - 画面上部のドロップダウンで「テキスト形式」または「Word形式」を選択

2. **ダウンロード実行**
   - 「選択したセクションをダウンロード」ボタンをクリック
   - ファイルが自動的にダウンロードされます

## ⚠️ トラブルシューティング

### よくある問題と解決方法

#### ファイルアップロードエラー

**問題**: 「サポートされていないファイル形式です」エラー
**解決方法**:
- ファイル拡張子が .txt または .docx であることを確認
- ファイル名に特殊文字が含まれていないか確認
- ファイルが破損していないか確認

**問題**: 「ファイルサイズが大きすぎます」エラー
**解決方法**:
- ファイルサイズを10MB以下に削減
- 不要な内容を削除
- ファイルを分割してアップロード

#### 処理エラー

**問題**: アップロード後に処理が止まる
**解決方法**:
- ページを再読み込み
- ファイル内容を確認（文字コードの問題の可能性）
- 別のファイルで試行

#### 表示エラー

**問題**: データが正しく表示されない
**解決方法**:
- ブラウザのキャッシュをクリア
- ページを再読み込み
- 別のブラウザで試行

### エラーメッセージ対応表

| エラーメッセージ | 原因 | 解決方法 |
|---|---|---|
| インターネット接続エラー | ネットワーク問題 | 接続確認後、再試行 |
| ファイルサイズエラー | ファイルが大きすぎる | ファイルサイズを削減 |
| ファイル形式エラー | 非対応の形式 | .txt/.docx形式を使用 |
| サーバーエラー | 一時的なサーバー問題 | 時間をおいて再試行 |

## 🎯 効率的な使用のコツ

### 1. ファイル準備
- **事前整理**: アップロード前にファイル内容を確認
- **命名規則**: 分かりやすいファイル名を使用
- **バックアップ**: 元ファイルのバックアップを保持

### 2. 編集作業
- **一括編集**: 話者名の統一は最初に実施
- **段階的除外**: 重要度の低いセクションから段階的に除外
- **時間確認**: 定期的に合計時間を確認

### 3. 品質管理
- **校正確認**: 編集後は内容を再確認
- **出力テスト**: 小さなセクションで出力テスト実施
- **バージョン管理**: 異なるバージョンをダウンロード保存
