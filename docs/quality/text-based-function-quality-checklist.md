## 🎯 テスト概要

このチェックリストは、テキストベース議事録機能の品質を確認するためのものです。各項目を順番に確認し、問題があれば記録してください。

## ✅ 基本機能チェック

### 1. ログイン・セッション管理
- [ ] **ログイン成功**: スムーズにログインできる
- [ ] **セッション作成**: 新しいセッションを作成できる
- [ ] **セッション一覧**: 過去のセッションが正しく表示される
- [ ] **セッション検索**: キーワードで検索できる

### 2. テキストファイルインポート
- [ ] **NOTTA形式**: .txtファイルが正常にインポートできる
- [ ] **MANUS形式**: .docxファイルが正常にインポートできる
- [ ] **ファイル選択**: ドラッグ&ドロップが機能する
- [ ] **プレビュー表示**: インポート前に内容確認できる
- [ ] **エラー表示**: 非対応ファイルで適切なエラーが出る

### 3. テキスト編集機能
- [ ] **話者名編集**: 「話者不明」を実際の名前に変更できる
- [ ] **内容編集**: テキスト内容を修正できる
- [ ] **セクション追加**: 新しいセクションを追加できる
- [ ] **セクション削除**: 不要なセクションを削除できる
- [ ] **セクション並び替え**: ドラッグで順序変更できる
- [ ] **自動保存**: 編集内容が自動的に保存される

### 4. エクスポート機能
- [ ] **Word形式**: .docxファイルでダウンロードできる
- [ ] **テキスト形式**: .txtファイルでダウンロードできる
- [ ] **フォーマット保持**: 話者名・タイムスタンプが正しく出力される
- [ ] **ファイル名**: 適切なファイル名で保存される

## 🚀 パフォーマンスチェック

### 応答速度
- [ ] **ページ読み込み**: 2秒以内に表示される
- [ ] **ファイルインポート**: 5秒以内に完了する
- [ ] **編集操作**: 遅延なく反応する
- [ ] **保存処理**: 1秒以内に完了する
- [ ] **エクスポート**: 3秒以内にダウンロード開始

### データ容量
- [ ] **小サイズ**: 100KB以下のファイル処理
- [ ] **中サイズ**: 1MB程度のファイル処理
- [ ] **大サイズ**: 5MB以上のファイル処理
- [ ] **セクション数**: 100セクション以上の処理

## 💡 ユーザビリティチェック

### 使いやすさ
- [ ] **直感的操作**: マニュアルなしで基本操作ができる
- [ ] **視覚的フィードバック**: 操作結果が明確にわかる
- [ ] **エラーメッセージ**: 問題発生時の対処法が明確
- [ ] **ヘルプ情報**: 必要な時に使い方を確認できる

### デザイン・レイアウト
- [ ] **見やすさ**: テキストが読みやすい
- [ ] **操作領域**: ボタンやリンクがクリックしやすい
- [ ] **色使い**: 目に優しく、重要な部分が強調されている
- [ ] **レスポンシブ**: 画面サイズに応じて適切に表示

## 🛡️ 信頼性チェック

### データ保護
- [ ] **編集内容保持**: ページ更新しても編集内容が残る
- [ ] **同時編集防止**: 他の人が編集中は適切に制御される
- [ ] **バックアップ**: 誤操作時の復元機能がある

### エラー処理
- [ ] **ネットワークエラー**: 接続切断時の適切な対応
- [ ] **入力エラー**: 不正な入力への適切な対応
- [ ] **システムエラー**: 予期しないエラーへの対応
